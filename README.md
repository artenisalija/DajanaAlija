# Dajana Hoti Website

Static one-page website prepared for GitHub Pages hosting.

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (or your default branch)
   - `Folder`: `/ (root)`
4. Save and wait for GitHub to publish.

Your site will be available at:
- `https://<username>.github.io/<repo>/` (project site), or
- `https://<username>.github.io/` (user site repo)

## Custom domain (optional)

If you want `https://dajanahoti.com`, add your domain in:
- `Settings` -> `Pages` -> `Custom domain`

Then configure DNS at your domain provider and enable HTTPS in GitHub Pages settings.
